# First Love — préparation App Store / Google Play

## Déjà prêt
- web app publique en ligne
- interface mobile/tablette
- PWA installable
- config Capacitor de base

## À faire avant publication Google Play
1. créer le compte Google Play Console
2. générer les icônes PNG officielles 512x512 + feature graphic
3. créer les captures d'écran téléphone
4. ouvrir Android Studio et lancer `npx cap add android` puis build
5. remplir fiche store, confidentialité et classification

## À faire avant publication App Store iPhone
1. compte Apple Developer
2. Mac + Xcode obligatoires pour build iOS
3. lancer `npx cap add ios` puis archive via Xcode
4. préparer captures iPhone + iPad
5. renseigner privacy nutrition labels

## Recommandation
Commencer par :
- publication web/PWA
- ensuite Android
- ensuite iPhone avec Mac/Xcode
